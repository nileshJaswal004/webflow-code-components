## [1.0.0] — 2026-05-19

### Added
- Initial boilerplate with 7 starter components: `Button`, `Badge`, `Alert`, `Countdown`, `FAQAccordion`, `InteractiveForm`, `Modal`
- `WebflowVariablesPage` — visual reference for all `--wf-*` CSS design tokens
- `DocumentationPage` — in-app step-by-step guide for the team
- Dashboard home page with Hogwarts / Marauder's Map theme
- Global design token system in `src/styles/webflow-variables.css`
